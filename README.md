## 功能说明
 - 整体架构：Fragment主导，viewpager + fragment， Material Design设计，MVP架构
 - 网络： Rxjava2 + Retrofit2 + OkHttp3
 - view注入： butterknife
 - 图片加载： glide
 - 事件管理： eventbus
 - debug调试：stetho
